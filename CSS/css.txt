15) What is RWD?
RWD stands for Responsive Web Design. This technique is used to display the designed page perfectly on every screen size and device, for example, mobile, tablet, desktop and laptop. You don't need to create a different page for each device.

16) What are the benefits of CSS sprites?
:CSS sprites can reduce the number of HTTP requests and improve website performance.

17) What is the difference between logical tags and physical tags?
:-Physical tags describe the appearance of content, while logical tags describe the purpose or meaning of content
Physical tags include tags like <b>, <i>, and <u>, while logical tags include tags like <header>, <nav>, and <section>.

18) What is the CSS Box model and what are its elements?
:-The CSS Box model is a layout concept that describes how elements on a webpage are structured, consisting of content, padding, border, and margin.

19) What is the float property of CSS?
:-The float property is used to position elements to the left or right within their parent container in CSS.

20) What is the purpose of the z-index and how is it used?
:-z-index controls the layering of elements on a webpage, allowing designers to determine which elements appear in front of others.

Z-Index can take the following values:

Auto: Sets the stack order equal to its parents.
Number: Orders the stack order.
Initial: Sets this property to its default value (0).
Inherit: Inherits this property from its parent element.

21) Explain the difference between visibility: hidden and display: none? 
:- "visibility: hidden" hides an element while keeping its space on the page, while "display: none" hides an element and removes its space on the page.
